di sini terdapat dua config startsip
1.untuk zsh 
2.untuk bash (starship-bash.toml)

semuanya berada pada folder ~/.config/starship


ini dibuat karena alacrity tidak mendukung rendering gambar, jadi kita pake terminal nya bash
agar berbeda dengan kitty yang menggunakan zsh, dan tema nya bisa dari starship atau ohmyzsh

bash menggunakan starship , yang di akali pada `export STARSHIPCONFIIG' nya path nya kita bedakan
, tetap dalam satu folder di starhsip tapi file .TOML yang berebeda 
dengan nama `starship-bash.toml` 

ini agar tidak ada conflic path, dan seetiap masing masing shell, baik bash atau zsh, export copnfig path nya berbeda nama 

jadi alacrity kombinasi dengan bash yang menggunakan starship

>note: saat mengganti config starship , pertahtikan jangan sampe meninpa path yang untuk zsh, secara defaulnya untuk zsh, jadi di bedakan
